Moedict-addon-app

moedict-app addon for moedict-desktop

Download this project as a .zip file Download this project as a tar.gz file

moedict desktop addon

這是基於 http://moedict.tw/ 之 moedict-app 離線查詢 App 改版而來。 本 Addon 需配合 moedict-desktop 使用。

Build

build.sh

本機運行

cd webapp && make # runs in http://127.0.0.1:8888/
